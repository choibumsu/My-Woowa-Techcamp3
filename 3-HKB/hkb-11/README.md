## ðŸ§žQuick Start

### 1. Clone & Install Packages
```bash
git clone https://github.com/woowa-techcamp-2020/hkb-11.git

cd hkb-11

npm install
```

### 2. Add .env
Add `.env` to root folder

```
DB_HOST= *.*.*.*
DB_USER= agrajak
DB_PASSWORD= ********
DB_DATABASE= hkb 
```

### 3. Run server
**Develop mode**
```bash
npm run dev
```

automatically open http://localhost:9000

**Production mode**
```bash
npm start
```

Then, you can access to your server http://localhost:3000
